# Master Plan

## Goal

Build a high-quality, agent-assisted web agency that can deliver productized websites for local businesses with strong margins, consistent quality, and human-reviewed client communication.

## Core Strategy

The business should be agent-first, human-approved.

Agents handle:

- research
- lead enrichment
- outreach drafting
- meeting preparation
- meeting summarization
- proposal drafting
- content planning
- project bootstrap
- QA support
- reporting and maintenance workflows

Human review remains responsible for:

- final outreach approval
- discovery calls
- pricing and scope approval
- design taste and direction
- launch approval
- trust-sensitive client communication

## Initial Market Direction

Primary target niches:

1. home services in the Greater Toronto Area

Secondary target niche after the first delivery system is working:

1. gyms and fitness studios in the Greater Toronto Area

Secondary niches for later expansion:

1. med spas and wellness clinics
2. dental and specialty clinics

Avoid starting with restaurants or ecommerce as the primary niche.

## Delivery Model

Use a productized service model built on:

- Next.js
- Payload CMS
- reusable block library
- design token system
- template families by niche
- tier-based feature configuration

The delivery engine should optimize for repeatability, not unlimited custom work.

## Initial Offer Structure

Three site tiers:

1. Launch
2. Growth
3. Premium

Recurring revenue after launch:

1. Care Plan
2. SEO / content support
3. conversion optimization / landing page iteration

## Build Philosophy

1. Start with one niche and one strong template family.
2. Build the starter stack before deep automation.
3. Add agents only where the workflow is repeatable and measurable.
4. Use dedicated design tools for visual exploration.
5. Use the operating system to coordinate agents through shared business records.
6. Treat code quality, security, and review rules as first-class operating constraints, not cleanup work.

## Success Criteria

The first version is successful when it can:

1. support real outreach
2. generate repeatable proposals
3. bootstrap projects from a standard stack
4. deliver agency-grade quality with human review
5. support at least one niche with a credible demo template and sales story
