# Metrics

Track these metrics from the start, even if initially by hand.

## Sales Metrics

- leads researched
- leads qualified
- outreach sent
- reply rate
- positive reply rate
- meetings booked
- meeting-to-proposal rate
- proposal acceptance rate
- average deal value

## Delivery Metrics

- average days to launch
- average revisions per project
- average custom work beyond standard tier
- QA issues found before launch
- QA issues found after launch
- project gross margin estimate

## Retention Metrics

- care plan attach rate
- support volume per client
- average time spent on support
- upsell conversion rate
- client referral rate

## Agent Quality Metrics

- outreach drafts needing major rewrite
- discovery summaries needing manual correction
- proposals needing major rewrite
- QA false positive rate
- number of human overrides to agent recommendations

## Notes

The goal is not perfect analytics at the beginning. The goal is to build enough measurement to know whether the operating system is helping or only feeling productive.
